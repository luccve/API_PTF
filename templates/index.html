Olá, meu nome é Estevão Lucas Ramos da Silva. Este é o meu trabalho de conclusão de curso (TCC) sobre PTFs.

Para fazer a requisão:


HTTP

POST /ptf HTTP/1.1 Host: 127.0.0.1:5000 Content-Type: application/json Content-Length: 113 { "sand": 30, "clay": 40, "silt": 30, "density": 1.5, "org_mat": 0.02, "porosity": 0.15 }


CURL

curl --location '127.0.0.1:5000/ptf' \ --header 'Content-Type: application/json' \ --data '{ "sand": 30, "clay": 40, "silt": 30, "density": 1.5, "org_mat": 0.02, "porosity": 0.15 }'


Javascript Fetch

var raw = JSON.stringify({ "sand": 30, "clay": 40, "silt": 30, "density": 1.5, "org_mat": 0.02, "porosity": 0.15 });
fetch("127.0.0.1:5000/ptf", { method: 'POST', headers: myHeaders, body: raw, redirect: 'follow' })
